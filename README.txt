Sistema de Gestión de Expedientes (SGE)
Trabajo Práctico - Fase 1

La solución fue desarrollada respetando la estructura solicitada de Arquitectura Limpia:

* SGE.Dominio
* SGE.Aplicacion
* SGE.Infraestructura
* SGE.Consola

Se implementaron:

* Entidades de dominio
* Value Objects
* Enumerativos
* Excepciones 
* Casos de uso
* DTOs Request/Response (incluyendo vacíos)
* Servicio de actualización automática de estado
* Repositorios TXT (Expedientes y Trámites)
* Servicio provisional de autorización
